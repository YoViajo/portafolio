# Portafolio - Soluciones analíticas y espaciales

Sitio de portafolio personal construido con Jekyll y GitHub Pages.

## Publicación

1. Crear repositorio en GitHub con nombre `tu-usuario.github.io` (para sitio principal) o cualquier nombre (para sitio de proyecto)
2. Subir estos archivos con GitHub Desktop
3. En GitHub → Settings → Pages → seleccionar rama `main` como fuente
4. El sitio estará disponible en `https://tu-usuario.github.io/`

## Estructura

```
├── _config.yml    # Configuración del sitio
├── index.md       # Contenido principal
└── README.md      # Este archivo
```

## Personalización

- Editar `_config.yml` para cambiar título, descripción y datos de contacto
- Editar `index.md` para modificar el contenido del portafolio
- El tema por defecto es `minima`; puede cambiarse en `_config.yml`
